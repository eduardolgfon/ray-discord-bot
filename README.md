# Ray Discord Bot

When I created this bot back in 2020, I didn't even know how to use Git yet. I created this repository to store the old, no longer maintained source code.

### Warning: you will find the worst programming practices in the world in this repository.

> I realized later that the event reloader has memory leaks and that it's possible to perform SQL injection by altering the profile description. Do not run this code in production; RAM consumption will increase significantly over time (this was the reason I ended the project).
